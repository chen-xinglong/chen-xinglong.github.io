# 那些年我看过的动漫

高中毕业到大学三年级之前，我看过不少动漫。  
其中大多是日本动漫。  
里面有好多动漫令我印象十分深刻。  
初看之时，十分惊艳；再看一遍，依然回味无穷。  
可惜的是，看过不久我就会忘掉，甚至连名字也一起。  
所以我想记录一下那些年我看过的动漫。  
从三年纪开始，不知为什么，我就不再看动漫了。  
现在是时候“盖棺定论”了。
对我那三四年间看过的动漫，进行最后的回忆与总结。  
这份记录，是为了更好的忘记。
