## 1.1

#### 1.1.1.3 汇编程序 

把汇编语言程序翻译成目标程序的过程称为汇编。  

### 1.1.3 恰当的使用汇编语言

- 汇编语言在“时空”方面最有效率

使用场景：
- 对软件的执行时间或存储容量有较高要求的场合。如：系统程序的关键核心，智能化仪器仪表的控制系统，实时控制系统等。
- 需要提高大型软件性能的场合。通常把大型软件中执行效率高的子程序（过程）用汇编语言编写，然后把他们与其他程序一起连接。
- 软件和硬件关系密切，软件要有直接和有效控制硬件的场合。如设备驱动程序等。
- 没有合适的高级语言的场合。

### 1.2.2 非数值数据的表示

#### 1. ASCII码（美国信息交换标准码，American Standard Code for Information Interchange）

- ASCII码是7位二进制编码
- 数字符号编码从30H开始、大写字母从41H开始，小写字母从61H开始
